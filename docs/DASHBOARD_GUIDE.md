## Getting Started

### First Time Login

When you first log in to Ecoleaf Cloud, you'll see:
- A personalized greeting based on time of day
- Dashboard overview with your statistics
- Navigation sidebar on the left
- Your profile information in the top right

### Navigation

The sidebar contains the following sections:
- **Dashboard** - Overview of your account
- **Upload** - Upload new files
- **My Files** - View and manage files
- **Storage** - Monitor storage usage
- **Analytics** - View usage statistics
- **Notes** - Create and manage notes
- **Text Storage** - Store text snippets
- **Settings** - Account settings

## Dashboard Overview

The Dashboard is your home screen, showing:

### Storage Usage Card
- **Current Usage**: Shows how much storage you've used (e.g., 1.25 GB / 10 GB)
- **Percentage Bar**: Visual representation of storage used
- **Remaining Storage**: How much space is left
- **Quick Stats**: Number of files, notes, and text items

### Statistics Cards
- **Total Files**: Number of files you've uploaded
- **Notes**: Number of notes you've created
- **Text Items**: Number of text snippets saved

### Recent Files
- Displays your 5 most recently uploaded files
- Shows file name, size, and upload date
- Click "View All" to see all files

## File Management

### Uploading Files

1. **Navigate to Upload Section**
   - Click "Upload" in the sidebar
   
2. **Select File**
   - Click the file input or drag & drop
   - Maximum file size: 50MB
   - Supported formats: All except executables (.exe, .bat, .sh)

3. **Upload Process**
   - Click "Upload File" button
   - Watch the progress bar (0-100%)
   - See success message when complete

**Tip**: You'll see the file name and size before uploading to confirm your selection.

### Viewing Files

1. **Navigate to My Files**
   - Click "My Files" in sidebar

2. **View Modes**
   - **Grid View**: Visual cards with file icons/thumbnails
   - **List View**: Compact list with file details
   - Toggle between views using icons at top right

### Grid View Features

**File Cards Display:**
- File icon or image thumbnail
- File name
- File size
- Upload date
- Hover to reveal action buttons:
  - **Preview**: View file in modal
  - **Download**: Download file to your device
  - **Delete**: Remove file permanently

**Image Files:**
- Automatic thumbnail preview
- Click preview to see full-size image in modal

### List View Features

**Compact Display:**
- All files in a scrollable list
- Quick access to actions
- Efficient for managing many files

**Actions Available:**
- **Eye Icon**: Preview file
- **Download Icon**: Download file
- **Trash Icon**: Delete file

### File Preview

**Supported Previews:**
- **Images**: Full-size modal view (PNG, JPG, GIF, etc.)
- **PDFs**: Embedded PDF viewer
- **Other files**: Download option shown

**Using Preview:**
1. Click the eye icon or file thumbnail
2. View file in modal window
3. Click X or outside modal to close
4. Download button available in preview

### Downloading Files

1. Click the download icon on any file
2. File will download with its original name
3. Check your browser's download folder

### Deleting Files

1. Click the trash icon on a file
2. Confirm deletion in the popup
3. File is permanently removed
4. Storage quota is updated automatically

**Warning**: Deletion is permanent and cannot be undone!

## Storage Management

### Storage Quota

**Your Plan:**
- Default: 10 GB total storage
- Storage is shared across:
  - Files
  - Notes (~5KB each)
  - Text snippets (~2KB each)

### Monitoring Usage

**Storage Section Shows:**

1. **Storage Quota Card**
   - Used vs. Total storage (e.g., 1.25 GB / 10 GB)
   - Percentage used with progress bar
   - Remaining storage
   - Breakdown by item type

2. **Storage Breakdown**
   - Storage by file category:
     - Images
     - Videos
     - Audio
     - PDFs
     - Documents
     - Other
   - Visual bars showing proportion of each type

### Managing Storage

**When Running Low on Storage:**

1. **Review Large Files**
   - Sort files by size (coming soon)
   - Consider deleting old/unused files

2. **Delete Unnecessary Items**
   - Old notes you no longer need
   - Duplicate files
   - Test uploads

3. **Download and Delete**
   - Download important files to your device
   - Delete from cloud to free up space

**Tip**: The dashboard always shows your current storage status at a glance.

## Analytics

The Analytics section provides insights into your usage patterns.

### Statistics Overview

**Top Row Stats:**
- **Total Files**: Count of all uploaded files
- **Total Storage**: Combined size of all files
- **Notes**: Number of notes created
- **Texts**: Number of text snippets

### File Types Distribution (Pie Chart)

**Shows:**
- Breakdown of files by category
- Percentage of each file type
- Color-coded segments

**Categories:**
- Images (Blue)
- Videos (Green)
- Audio (Purple)
- PDFs (Red)
- Documents (Orange)
- Other (Gray)

**How to Read:**
- Hover over segments for details
- Larger segments = more files of that type
- Percentages show proportion of total

### Upload Trends (Area Chart)

**Shows:**
- Storage uploaded over last 30 days
- Smooth wave/area visualization
- Dates on X-axis, storage size on Y-axis

**How to Read:**
- Peaks show days with high upload activity
- Hover over points for exact values
- Trend line shows overall pattern

**Use Cases:**
- Track your usage patterns
- Identify busy periods
- Plan storage upgrades

### Upload Activity (Bar Chart)

**Shows:**
- Number of files uploaded per day
- Last 30 days of activity
- Bars represent file count

**How to Read:**
- Taller bars = more files uploaded that day
- Hover for exact file count
- See which days you're most active

**Insights:**
- Understand your upload habits
- Track project timelines
- Monitor team activity (future feature)

## Notes

Notes are perfect for quick thoughts, meeting notes, or any text you want to keep.

### Creating a Note

1. **Navigate to Notes Section**
   - Click "Notes" in sidebar

2. **Fill in Note Form**
   - **Title**: Give your note a descriptive title
   - **Content**: Write your note (supports multi-line text)
   - Click "Save Note"

3. **Note is Created**
   - Appears immediately in "All Notes" section
   - Timestamp shows when created

### Viewing Notes

**Card Layout:**
- Notes displayed in grid of cards
- 2-3 columns depending on screen size
- Each card shows:
  - Title
  - Preview of content (first few lines)
  - Last updated timestamp
  - Delete button (top right)

**Reading Full Note:**
- Content is visible in the card
- Scroll within card to read full content
- Click anywhere on card to focus

### Editing Notes

**Current Version:**
- Notes are view-only after creation
- To update: Delete old note and create new one

**Future Update:**
- Edit button will be added
- In-place editing capability

### Deleting Notes

1. Click trash icon on note card (top right corner)
2. Confirm deletion
3. Note is permanently removed

**Tip**: Copy important content before deleting!

### Use Cases for Notes

- Meeting notes
- Quick reminders
- To-do lists
- Ideas and brainstorming
- Research notes
- Personal journal entries

## Text Storage

Text Storage is designed for longer text content, code snippets, or any text you want to store.

### Saving Text

1. **Navigate to Text Storage**
   - Click "Text Storage" in sidebar

2. **Fill in Text Form**
   - **Title**: Descriptive title (e.g., "Python Script", "Important Config")
   - **Content**: Paste or type your text
     - Supports very long text (10,000+ characters)
     - Preserves formatting and line breaks
   - Click "Save Text"

3. **Text is Saved**
   - Appears in "Saved Texts" list
   - Timestamp recorded

### Viewing Text Snippets

**List Layout:**
- Texts displayed in stacked cards
- Each card shows:
  - Title (large, bold)
  - Timestamp
  - Full content in monospace font
  - Action buttons (Copy, Delete)

**Reading Text:**
- Content shown in scrollable box
- Monospace font for code/structured text
- Preserves all formatting, indentation, line breaks

### Copying Text

1. Click the **Copy** icon (top right of text card)
2. Text is copied to your clipboard
3. Success message confirms copy
4. Paste anywhere (Ctrl+V / Cmd+V)

**Great for:**
- Code snippets
- Configuration files
- Long passwords or keys
- Structured data

### Deleting Text

1. Click trash icon on text card
2. Confirm deletion
3. Text is permanently removed

### Use Cases for Text Storage

- **Code Snippets**: Save reusable code blocks
- **Configuration Files**: Store config templates
- **API Keys**: Keep important keys (but not too sensitive!)
- **Documentation**: Store markdown or plain text docs
- **Scripts**: Save shell scripts, SQL queries
- **Logs**: Keep important log excerpts
- **Templates**: Email templates, document templates

**Security Note**: Don't store highly sensitive data like passwords or private keys. Use a dedicated password manager for those.

## Settings

Customize your account and preferences.

### Profile Information

**Display Name:**
- This name appears in your greeting ("Good morning, [Name]!")
- Change anytime
- Saves immediately

**How to Update:**
1. Navigate to Settings
2. Type new name in "Display Name" field
3. Click "Save Changes"
4. See success message
5. Refresh to see new name in greeting

### Account Information

**View Only:**
- **Email**: Your registered email address
- **Phone Number**: Your phone number (if registered via phone)
- These fields are read-only for security

### Account Actions

**Logout:**
1. Click red "Logout" button
2. Session ends immediately
3. Redirected to login page
4. Token is cleared from browser

**Security Note**: Always logout when using shared computers!

## Tips & Best Practices

### Storage Management

1. **Regular Cleanup**
   - Review files monthly
   - Delete old test uploads
   - Remove duplicate files

2. **Organize with Naming**
   - Use descriptive file names
   - Include dates: `report_2025-02-05.pdf`
   - Use categories: `work_proposal.docx`

3. **Monitor Usage**
   - Check storage section weekly
   - Set personal limits (e.g., "I'll keep usage under 8GB")
   - Download and archive old files

### File Uploads

1. **File Naming**
   - Use clear, descriptive names
   - Avoid special characters
   - Include version numbers if needed

2. **File Types**
   - Upload PDFs for documents (smaller than Word docs)
   - Compress large images before uploading
   - Use appropriate formats

3. **Upload Strategy**
   - Upload one file at a time for stability
   - Check file size before uploading
   - Wait for success message before closing page

### Notes Organization

1. **Descriptive Titles**
   - Use titles that explain content
   - Include dates for time-sensitive notes
   - Use categories: "[Work] Meeting Notes"

2. **Keep Notes Current**
   - Delete outdated notes
   - Archive important notes elsewhere
   - Review notes regularly

3. **Formatting Tips**
   - Use line breaks for readability
   - Structure with bullet points
   - Keep notes concise

### Text Storage

1. **Title Your Snippets**
   - Use clear titles: "React Component Template"
   - Include language: "Python - API Helper"
   - Add versions: "Config v2.1"

2. **Organize Content**
   - Add comments in code snippets
   - Include usage instructions
   - Document parameters

3. **Security**
   - Don't store passwords
   - Mask sensitive data
   - Use environment variables for keys

### Security Best Practices

1. **Password Management**
   - Use strong password
   - Don't share credentials
   - Change password periodically

2. **Session Management**
   - Logout when done
   - Don't stay logged in on public computers
   - Clear browser cache on shared devices

3. **File Privacy**
   - Don't upload confidential documents
   - Sensitive files should be encrypted
   - Review file permissions (future feature)

### Performance Tips

1. **Browser**
   - Use modern browsers (Chrome, Firefox, Edge)
   - Keep browser updated
   - Clear cache if issues occur

2. **Network**
   - Use stable internet for uploads
   - Avoid uploading on slow connections
   - Resume uploads if interrupted (future feature)

3. **Storage**
   - Keep under 80% storage usage
   - Regular cleanup improves performance
   - Archive old files locally
